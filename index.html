Connect to OneDrive (Excel)
Sync logs and projects from your Excel file stored on OneDrive or SharePoint. You need a free Azure app registration — takes 5 minutes.
1
Go to portal.azure.com → Azure Active Directory → App registrations → New registration. Name it anything, set Redirect URI (SPA) to this page's URL.
2
In the app → API permissions → Add → Microsoft Graph → Delegated → Files.ReadWrite → Grant admin consent.
3
Copy the Application (client) ID from the Overview page.
4
Open your Excel file in OneDrive / SharePoint → copy the URL from the browser. Paste the full OneDrive/SharePoint URL below — the file ID will be extracted automatically.
💡 The Excel file must have sheets named Logs and Projects with the column headers from the provided template. Sign-in uses Microsoft OAuth — no passwords are stored here.
